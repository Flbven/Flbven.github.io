变量和基本类型

# 第二章 变量和基本类型

# 1.1 常用内置类型
类型 含义 最小尺寸
bool 布尔类型 undefined
char 字符类型 8位
int 整型 16位(在32位系统中为32位)
long 长整型 32位
long long 长整形 64位
float 单精度浮点数 6位有效数字
double 双精度浮点数 10位有效数字

选择数据类型的方法：

  • 明确数值不可能为负值时， 选用无符号类型
  • 使用int执行整型运算，short太短而long一般和int有一样的尺寸（只有long>=int的规定）， 范围不够时选用long long
  • 算术表达式中不要使用char或bool
  • 执行浮点数运算用double
# 1.2 类型转换

  • 非布尔类型 -> 布尔类型 0 ？ false : true

  • 浮点数 -> 整数类型 保留小数点前的部分

  • 赋给无符号数一个超出表示范围的值时，结果是初始值对无符号类型表示数值总数取模后的余数，如将-1赋给8比特大小的数据类型结果为255

    所以在类似

    for(int i = 10 ; i >= 0 ; i -- )

    因为当i=0时，执行**i--**后无符号数并不会小于0，反而会变成更大的一个正数。

    ​ 最好不要混用带符号类型和无符号类型。

要避免无法预知和依赖于实现环境的行为，比如如果我们把int的尺寸看成是一个确定不变的已知值，那么这样的程序就被称为不可移植的。当程序移植到别的机器上时，依赖于实现环境的程序就可能发生错误。

# 1.4 变量
# 变量的定义

  • 定义形式： 类型说明符，随后紧跟一个或多个变量名所组成的列表，定义时也可为一个或多个变量赋值。

  • 初始化: 当对象在创建时获得了一个特定的值， 我们说它被初始化了。

    • 默认初始化：如果定义时没有指定值，则它会被赋予一个默认值，默认值由定义类型决定。

      为了安全起见，建议初始化每一个内置类型的变量。

    • 列表初始化： 无论是初始化对象还是赋新值， 都可以用一组由花括号括起来的初始值。

      // 以下均为有效初始化
int x = 0;
int x = {0};
int x{0};
int x(0);

初始化与赋值并不相同， 初始化的含义是在创建变量的时候赋予其一个初始值， 而赋值的含义是用新值代替当前的旧值。

# 变量的声明与定义

为了支持分离式编译，C++将声明与定义区分开来， 声明使得名字为程序所知，定义负责创建与名字关联的实体。

如果想仅仅声明一个变量而非定义它，就需要使用关键字extern，而且不要显式地初始化变量。

extern int i;     //声明而非定义i
int j;            //声明并定义j
extern int i = 0; //extern失效, 变为定义 
//如果在函数体内部初始化一个标记为extern的变量	将引发错误
